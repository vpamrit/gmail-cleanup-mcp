✉ Gmail Cleaner

One-time setup

Gmail Cleaner talks to your mailbox through Google's own API, so you create a free personal credential once. Takes about five minutes.

  1. Create a Google Cloud project. Open the console ↗ and create a project (any name, e.g. gmail-cleanup).
  2. Enable the Gmail API. Open Gmail API ↗ and click Enable.
  3. Set up the OAuth consent screen. Open consent screen ↗: choose External, fill in the app name and your email, and on the Test users step add your own Gmail address. (Test mode is fine — no Google review needed for personal use.)
  4. Create the credential. Open Credentials ↗ → Create Credentials → OAuth client ID → application type Desktop app → Create → Download JSON.
  5. Drop that JSON below.
